Fuel Availability

This is a full-stack web application, developed single-handedly in response to the fuel crisis in Sri Lanka. The platform enables users to check fuel availability at local stations, while registered users can update the fuel status. The app also features a fully functional advanced admin panel and is free to use, open-source software released under the MIT License.

Papol Linux

This is a visually stunning Xubuntu-based respin distribution that incorporates the Xfce desktop environment to emulate the appearance of MacOS. This distribution is designed for ease-of-use. This offers a seamless blend of style and functionality, making it the perfect choice for those who want the aesthetic appeal of MacOS with the power and versatility of Linux.

InstaStalker

InstaStalker is an easy-to-use, beginner-friendly OSINT tool for uncovering publicly available information hidden within the Instagram user interface. The tool is simple to set up and provides a web server with a user-friendly interface for conducting OSINT research. InstaStalker covers a wide range of functionalities from the popular Instaloader library in Python, making it a comprehensive and convenient solution for obtaining information from Instagram.
