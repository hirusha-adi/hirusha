## Syntax

- `Set Count to 1`
- `Set Total to Total + Value` or `Add Value to Total`
- `Increment Count`
- `Decrement Count`
- `ReBoort()` - Procedure - inside double bar box
- `Set Flag to Check()` - Function - inside double bar box
